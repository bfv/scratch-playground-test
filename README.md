# scratch playground test

This project is to test Arno van der Ende's Scratch Playground VSCode extension.<br/>
See: [https://github.com/vdende/vscode-scratch-playground-public](https://github.com/vdende/vscode-scratch-playground-public)

IMPORTANT: put you linux development server license in `progress.cfg` in `oe-license/`. If any other place is necessary, modify `.devcontainer/.env`:
```
PROGRESS_CFG=../oe-license/progress.cfg
```
